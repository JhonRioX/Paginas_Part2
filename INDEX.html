Questionário


Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

R: Em um ambiente de hospedagem, o servidor web serve para entregar arquivos estáticos como HTML, CSS e imagens. Um exemplo de servidor web é o Apache.
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

R: Leve e eficiente. Ele é muito utilizado em projetos de alto desempenho por causa do baixo consumo de recursos e sua capacidade de balanceamento de carga e proxy reverso.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

R: O protocolo HTTP é para transferência de páginas web (sem criptografia). Já o HTTPS é uma versão segura com criptografia SSL/TLS. Para aplicações modernas, o HTTPS é recomendado a fim de garantir segurança, privacidade e integridade nas comunicações da web.
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

R: As portas em uma rede de computadores servem para identificar serviços específicos em um dispositivo, permitindo a comunicação entre sistemas. A porta 443 é de suma importância para o tráfego seguro via HTTPS, garantindo conexões criptografadas.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

R: O endereço IP é um número identificador de um dispositivo na rede. Ele é crucial para roteamento e localização de dispositivos na internet ou redes locais.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

R: Hospedagem VPS é um servidor virtual com mais controle e desempenho. Ela é indicada para sites e aplicações que necessitam de mais controle, desempenho e segurança que a hospedagem compartilhada, mas sem o custo de um servidor dedicado.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

R: O Caddy é um servidor moderno com HTTPS automático e configuração simples.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

R: O servidor de aplicação mais indicado para esta situação é o ASP.NET Core, pois é uma plataforma moderna da Microsoft para web apps e APIs.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

R: O cliente envia uma requisição HTTP ao servidor, que, por sua vez, processa a solicitação (como acessar uma página) e retorna a resposta, geralmente na forma de conteúdo HTML, JSON ou arquivos estáticos.
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

R: O Apache Tomcat é um servidor de aplicação que implementa servlets e JSP. Ele é mais indicado para aplicações Java.
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

R: Um servidor web entrega arquivos como páginas e imagens para o navegador. Já o servidor de aplicação processa informações e gera respostas dinâmicas, como interações com bancos de dados e conteúdos personalizados.
Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

R: Um domínio na internet é o nome legível usado para identificar um site ou serviço online (exemplo: www.google.com). Ele é relacionado a um endereço IP por meio de um sistema de DNS, que traduz o nome de domínio em um número de IP, permitindo que os usuários acessem o site de forma mais fácil.
Questão 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

R: A hospedagem em nuvem é distribuída, escalável e com alta disponibilidade.
Questão 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

R: GlassFish = suporta Java / Node.js = suporta JavaScript.
Questão 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

R: Atualmente, a utilização de HTTPS em sites e sistemas é essencial para garantir a segurança das informações trocadas, como criptografia e proteção contra ataques. O HTTP expõe dados a interceptação e manipulação, colocando a privacidade e a integridade dos usuários em risco.
Questão 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

R: Se um site recebesse tráfego alto e precisasse de mais controle sobre configurações e recursos, uma hospedagem VPS seria mais vantajosa que uma hospedagem compartilhada, que oferece menos desempenho e flexibilidade. A VPS permite maior personalização e estabilidade.
Questão 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

R: O IP identifica de forma única um dispositivo em uma rede, permitindo sua localização e comunicação. A porta define o serviço específico dentro desse dispositivo, direcionando o tráfego para a aplicação ou processo correto.
Questão 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

R: É fundamental que um servidor web consiga lidar com HTTPS para garantir a segurança da comunicação, criptografando os dados trocados entre o cliente e o servidor. Isso protege contra interceptações e ataques.
Questão 19: Cite dois exemplos de servidores web e explique suas principais características.

R: Microsoft IIS = integrado ao Windows Server; LiteSpeed = focado em performance e compatível com Apache.
Questão 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

R: A hospedagem compartilhada é um tipo de serviço onde múltiplos sites dividem os mesmos recursos de um servidor, como CPU, memória e espaço em disco. Ela é indicada para sites pequenos ou de baixo tráfego, devido ao custo mais baixo, mas com menos controle e desempenho.
Questão 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

R: Um servidor de aplicação executa lógica de negócio e processa requisições dinâmicas, como interações com banco de dados. Já um servidor de arquivos estáticos apenas entrega conteúdos fixos, como imagens, CSS e HTML, sem processamento adicional.
Questão 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

R: Django. Ele funciona como um framework para Python com servidores como Gunicorn ou uWSGI.
Questão 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

R: WildFly é um servidor de aplicação e ele é indicado para a plataforma Java.
Questão 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

R: A porta 21 é usada pelo protocolo FTP (File Transfer Protocol), que permite a transferência de arquivos entre um cliente e um servidor. Ela é a porta de controle para a comunicação entre os dois sistemas.
Questão 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

R: O servidor LiteSpeed é compatível com o Apache porque ele pode interpretar arquivos de configuração do Apache e usa a mesma estrutura de módulos, permitindo que sites e aplicações que funcionam no Apache sejam migrados para o LiteSpeed sem grandes alterações. As vantagens dessa compatibilidade incluem maior desempenho, maior capacidade de gerenciamento de tráfego e recursos de segurança aprimorados, mantendo a facilidade de configuração do Apache.
Questão 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

R: Microsoft IIS é um servidor web da Microsoft, integrado ao Windows Server. O IIS é comumente adotado em situações onde a infraestrutura da organização já é baseada em Windows, especialmente em ambientes corporativos que utilizam tecnologias como o ASP.NET, SQL Server, ou outras soluções da Microsoft.
Questão 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

R: O Node.js é uma plataforma de desenvolvimento que permite executar JavaScript no servidor, ideal para aplicações escaláveis e de alta performance. Ele é amplamente utilizado em APIs, sistemas em tempo real, como chats e jogos online, e microserviços.
Questão 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

R: O GlassFish é um servidor de aplicações Java que implementa a especificação Java EE (Enterprise Edition), oferecendo suporte a tecnologias como EJB, JSP, e servlets. Ele é utilizado para desenvolver e hospedar aplicações corporativas baseadas em Java.
Questão 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

R: Puma e Unicorn são servidores web para Ruby, usados principalmente em aplicações Ruby on Rails. O Puma é otimizado para múltiplos núcleos, enquanto o Unicorn é focado em alto desempenho com processos isolados.
Questão 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

R: A escolha entre hospedagem local, compartilhada, VPS e nuvem impacta diretamente a escalabilidade, performance, custo e segurança do projeto. Uma escolha errada pode afetar a experiência do usuário e a gestão de recursos ao longo do tempo.
